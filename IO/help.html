Help

Syntax

The syntax for this is keywords left to right.
{move/go/walk/travel} {north/south/east/west}.
Open Chest
Most commands can be called several different ways so get creative.
All actions cost Time Units. These will only really affect you during time sensative actions like combat.

System Commands

& - Repeat the last recieved message.
- Show last message before entering the Glossary.
- Save your Session.
- Lookup an item in the . Most highlighted items can be looked up.

- Compendium Lookup. [Requires Spellbooks]

AutoFills

This uses autofills and auto commands. These will display slightly darker and you can see what behavior it will have based on your cursor.
Clicking will immidiately send the text clicked while clicking will add it to your input box.
